# Practice 9 - Game Development with Python
## 🎮 Description
This project contains three simple games created using Python:
- Moving Ball Game
- Music Player (Console-based)
- Clock Application
The purpose of this practice is to learn basic game development concepts such as:
- Handling user input (keyboard control)
- Working with loops and events
- Basic graphics using Turtle
- Program structure and organization

## 🎯 Features
### 1. Moving Ball Game
- Red ball controlled by arrow keys
- Movement step: 20 pixels
- Boundary checking implemented
### 2. Music Player
- Console-based player
- Next / Back / Play / Quit controls
- Playlist simulation
### 3. Clock Application
- Displays real-time system clock
- Updates every second
- Uses Python time module
